#  Student Performance Analysis Dashboard ‚Äì Power BI

##  Project Overview
This project analyzes academic performance of 1,000 students in Math, Reading, and Writing using Power BI.  
The dashboard highlights how factors such as gender, race/ethnicity, parental education, lunch type, and test preparation influence student results.

##  Dataset
The dataset includes the following fields:

- test preparation course

- reading score

##  Key Insights
- Female students score higher in Reading and Writing  
- Students with *standard lunch* perform better  
- Test preparation course improves performance  
- Higher parental education correlates with higher scores  
- Group E students have the best performance
